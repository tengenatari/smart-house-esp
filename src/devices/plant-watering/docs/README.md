#### Взаимодействие с сервером

**Формат запросов:**

HTTP POST

URL: /api/v1/heartbeat/PLANT-WATERING-1

Пример тела запроса:
```
{
    "token": "токен",
    "values": {
        "soil_raw": 3619,
        "moisture_percent": 0,
        "pump": 0,
        "mode": 1,
        "last_watering_ms": 40589,
        "sensor_warning": 0,
        "dry_auto_count": 1,
        "flow_rate": 0.0
    }
}
```

**Формат ответов:**

HTTP 200 OK

**Пример ответа:**

{
    "message": "OK",
    "trigger_active": false,
    "state": 0.0,
    "metadata": {}
}


#### Описание работы системы:

**Измерение влажности почвы:**

- Влажность почвы считывается с аналогового датчика почвы. Значение влажности в процентах вычисляется с помощью линейной формулы, используя считанное значение от 0 до 4095:

- moisture_percent = map(soil_raw, 3620, 1680, 0, 100)

**Автополив:**

- Полив включается при влажности ≤ 35%

- Полив прекращается, если влажность ≥ 55% или по достижении потока воды 100 единиц воды

**Принудительный полив:**

- Если не было полива в течение 7 минут, выполняется принудительный полив на 7 секунд до достижения потока 100 единиц воды

**Ручной полив:**

- Полив доступен через кнопку и длится 5 секунд

**Защита от повторного полива:**

- Следующий полив по сухости почвы не раньше, чем через минуту после прошлого полива

- Каждый такой подозрительный полив раньше чем через 1 минуту считается и при достижении 3 подозрительных поливов на сервер отправляется предупреждение

- Как только между одним из подозрительных поливов влажность нормализовалась/прошло больше минуты, счетчик сбрасывается до 0
