#### Описание

Для автоматического управления шторами и связи с облачным сервером используется микроконтроллер ESP-32.

В системе используются следующие компоненты:

- Шаговый двигатель - подключен к пинам 14, 27, 26, 25 ESP-32. Обеспечивает перемещение штор.

- Концевики (2 шт) - подключены к пинам 16 (полностью закрыто) и 17 (полностью открыто) с подтяжкой к питанию. Служат для калибровки и определения крайних положений.

- Кнопки управления (2 шт) - зеленая (открыть) на пине 13, красная (закрыть) на пине 12. Используются для ручного управления.

- Переключатель режимов - подключен к пину 15, позволяет выбрать ручной или автоматический режим управления.

- Индикатор здоровья системы - встроенный светодиод на пине 2, горит при наличии подключения к Wi-Fi.

Система поддерживает два режима работы:

- Ручной режим - управление осуществляется кнопками "Открыть" и "Закрыть"

- Автоматический режим - позиция штор задается командами с сервера

При запуске система автоматически определяет крайние положения штор с помощью концевиков (в симуляции нужно нажать на кнопки) и вычисляет полный диапазон перемещения в шагах. Текущее положение штор отслеживается в процентах (0% - полностью закрыто, 100% - полностью открыто).

#### Взаимодействие с сервером

Микроконтроллер подключается к Wi-Fi сети и обменивается данными с сервером. В автоматическом режиме сервер может задавать целевое положение штор.

- В ручном режиме (mode: "manual"): система игнорирует поле state из ответа сервера, управление осуществляется только кнопками.

- В автоматическом режиме (mode: "auto"): при получении ответа с trigger_active: true и значением state, система автоматически перемещает шторы в указанную позицию.

Отправка состояния на сервер происходит:

- В автоматическом режиме - каждые 10 секунд для проверки команд с сервера

- В ручном режиме - каждые 10 секунд, если были изменения положения (объединяя все изменения за период)

- При смене режима работы - при переключении между ручным и автоматическим режимом

**Формат запросов:**

HTTP POST 

URL: /api/v1/heartbeat/CURTAINS-1

body:
```
{
    "token": "curtains_token",
    "values": {
        "position": 75,
        "mode": "auto"
    }
}
```

**Формат ответов:**

HTTP 200 - подтверждение получения данных 

```
{
    "message": "OK",
    "trigger_active": true,
    "state": 100.0,
    "metadata": {}
}
```
